当前仓库已经从“分层草图”进入“边界正在被代码和测试共同守护”的阶段。

当前最稳定的事实：

1. 四层结构已经是唯一主线。
2. `Analysis.Engine` 已经进入 `Domain + Logic` 组合表达，不再以旧 `src/Analysis` 作为主实现区。
3. `Application` 当前以 AppService 编排壳为主。
4. `RewriteWorkflow` 已拆成稳定阶段组合。
5. `RewriteDecision / RewritePlan / RewriteResult / VerificationEvidence / RunReport / ChangeCandidate` 的关键不变量已持续回收进 Domain。
6. `RewriteWorkflowArtifactAssembler` 当前保持单一装配入口，五段阶段边界已经显式化。
7. `WorkflowEventSequenceBuilder + IDomainEventRecorder + RewriteWorkflowStageResults` 当前已形成稳定事件收尾链。
