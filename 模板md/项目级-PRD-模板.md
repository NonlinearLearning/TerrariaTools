# 项目级 PRD 模板

> 用途：定义为什么做、给谁做、要做到什么程度。
>
> 写法信号来源：
> - React / Next.js / FastAPI：先讲用户入口、快速开始、学习路径，再展开细节
> - Kubernetes / TypeScript：明确区分目标、非目标、概念、任务、参考
> - Rust RFC / Vue RFC / Electron RFC：重大变更必须写清动机、影响、替代方案
